# Learning Flask 🚀

Welcome to my Flask learning repository! This project serves as a playground where I'm documenting my journey into web development with Python and Flask.

## 📁 Project Structure

This repository is organized to show my progress:

- **[proyectos/](./proyectos)**: This folder contains various small projects, exercises, and experiments I've built while learning different Flask concepts.

## 🛠️ Tech Stack

- **Backend**: Python & Flask
- **Database**: SQLAlchemy (ORM)
- **Authentication**: Flask-Login
- **Security**: Password hashing with Werkzeug
- **Frontend**: HTML5 & CSS (Jinja2 Templates)

## ✨ Features implemented so far:

- [x] User Registration & Login
- [x] Role-based access (Admin & User)
- [x] Full CRUD for blog posts
- [x] Interactive comment system
- [x] Flash messaging for user feedback
- [x] Custom decorators for route protection
